# Programa-Para-Consult-rio-Psicol-gico-v1.2
Um sistema desktop em Python com CustomTkinter, voltado à gestão de pacientes e relatórios pós-atendimento em consultórios de psicologia. Focado em organização, segurança e praticidade, oferece interface intuitiva para registrar dados clínicos, garantindo confidencialidade e eficiência profissional.
